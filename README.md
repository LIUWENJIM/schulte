# 舒尔特方格 · Schulte Table

一个简洁的注意力训练工具，基于经典的舒尔特方格方法。

## 功能

- 5 种网格大小：3×3 / 4×4 / 5×5 / 6×6 / 7×7
- 实时计时与错误记录
- 五级评分系统（S / A / B / C / D）
- 训练历史记录与统计
- 移动端响应式设计

## 评分标准

| 网格 | 目标时间 |
|------|---------|
| 3×3  | 8 秒    |
| 4×4  | 15 秒   |
| 5×5  | 25 秒   |
| 6×6  | 40 秒   |
| 7×7  | 60 秒   |

**计分公式：** 基础分 + 时间奖励 - 错误惩罚

| 等级 | 分数 | 含义 |
|------|------|------|
| S    | ≥150 | 大师级 |
| A    | 120+ | 优秀  |
| B    | 90+  | 良好  |
| C    | 60+  | 一般  |
| D    | <60  | 需加强 |

## 本地运行

```bash
npm install
npm run dev
```

## 技术栈

- React 18 + TypeScript
- Vite
- Tailwind CSS v4
- React Router
- localStorage 本地存储
